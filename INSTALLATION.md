# Installation - SPEC-1

## Prerequisites

- VS Code or Cursor.
- GitHub Copilot enabled.
- Access to Copilot Chat and Plan Mode.

No runtime environment is required for this exercise.

## Setup

1. Open the `repo/` folder.
2. Read `repo/user-story.md`.
3. Create `repo/spec.md`; this is where the generated specification will be saved.

## Verification

There is no command to run. The exercise is complete when `repo/spec.md` exists and contains a structured functional specification.
