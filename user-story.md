# TaskFlow API - User Story

As a developer, I need a REST API to manage tasks.

A task has:

- a title
- an optional description
- a priority: low, medium, high
- a status: todo, in_progress, done
- a due date

I need to:

- create tasks
- read tasks
- update tasks
- delete tasks
- list tasks with optional filters on status and priority

The API should use JSON and return appropriate HTTP status codes.
